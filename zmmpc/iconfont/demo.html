第一步：使用font-face声明字体 
@font-face {font-family: 'esuniconfont';
    src: url('iconfont.icon-eot'); /* IE9*/
    src: url('iconfont.icon-eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('iconfont.icon-woff') format('woff'), /* chrome、firefox */
    url('iconfont.icon-ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.icon-2+*/
    url('iconfont.icon-svg#esuniconfont') format('svg'); /* iOS 4.icon-1- */
}
第二步：定义使用iconfont的样式 
.icon-esuniconfont{
    font-family:"esuniconfont" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.icon-2px;
    -moz-osx-font-smoothing: grayscale;}
第三步：挑选相应图标并获取字体编码，应用于页面 
<i class="esuniconfont">&#xe604;</i>
或者 
<i class="esuniconfont icon-gantanhao"></i>
